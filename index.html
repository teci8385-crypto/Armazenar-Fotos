Armazenamento na Nuvem — Fotos & Vídeos (100000 GB)

Interface client-side — conectar a um backend para enviar aos provedores (S3 / GCS / Azure)
Arraste e solte fotos e vídeos aqui
Ou clique para selecionar arquivos (suporta upload em lote, até o seu limite contratado — ex: 100000 GB)
Dica: este é apenas o frontend. Para armazenar realmente na nuvem (100000 GB = 100 TB) você precisa configurar um bucket no provedor desejado (AWS S3 / Google Cloud Storage / Azure Blob). Veja instruções comentadas no topo do arquivo HTML (linhas de comentário) para exemplos de backend e segurança (presigned URLs, políticas CORS, limites e faturamento).
